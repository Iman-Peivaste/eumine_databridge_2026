## Model Architecture

ALIGNN (fine-tuned from JARVIS pretrained weights) +
MACE-MP-0 (universal potential, Materials Project pretrained) +
Optuna-optimized weighted ensemble +
Isotonic calibration layer.

```
CIF Structure
     │
     ├── ALIGNN EF (4 layers) ──┐
     ├── ALIGNN BG (6 layers) ──┤
     ├── MACE-MP-0 EF ──────────┤ Weighted Ensemble → Calibration → Prediction
     └── MACE-MP-0 BG head ─────┘
```

**Data sources:** Materials Project · JARVIS-DFT · OQMD

## Quickstart — Reproduce Predictions

### 1. Environment

```bash
conda activate ip
pip install -e .
```

### 2. Download data

```bash
python scripts/download_data.py
cd data/raw
unzip bridge_dataset_train_structures.zip -d train_structures/
unzip bridge_dataset_val_structures.zip   -d val_structures/
unzip test_input_structures.zip           -d test_structures/
```

### 3. Set API keys

```bash
cp .env.example .env
# Edit .env and add your Materials Project API key
```

### 4. Fetch external data and fit harmonizer

```bash
python scripts/fetch_external_data.py
```

### 5. Train models (optional — pretrained weights available)

```bash
# Train on train split only (Step 3)
python scripts/train_alignn.py --target both

# Build ensemble (Step 4)
python scripts/run_ensemble.py

# Full retrain on train+val (Step 4B — final submission)
python scripts/retrain_full.py
```

### 6. Run MatFed compliance tests

```bash
bash scripts/run_matfed_compliance.sh
# Expected: 5/5 tests pass
```

### 7. Generate submission

```bash
# predictions_test.json is already generated by retrain_full.py
# To regenerate manually:
python scripts/generate_submission.py
```

## MatFed API v1 Usage

```python
import sys
sys.path.insert(0, 'src')
sys.path.insert(0, '../hackathon_ref/matfed-api-template')

from eumine_databridge.matfed.predictor import LISTEuMINePredictor
from pymatgen.core import Structure

predictor = LISTEuMINePredictor()
predictor.load_model('models/full_retrain')

structures = [Structure.from_file('path/to/material.cif')]
results = predictor.predict(structures)

# Each result contains:
# {
#   'formation_energy_per_atom': float,  # eV/atom
#   'band_gap': float,                   # eV
#   'model_id': str,
#   'data_sources_used': list[str],
#   'uncertainty_ef': float,
#   'uncertainty_bg': float
# }
```

## Compute Requirements

- CPU-compatible: Yes (GPU strongly recommended for training)
- GPU used: NVIDIA RTX A4000, 16 GB VRAM
- Training time: ~15 min per ALIGNN model (700 samples, 300 epochs)
- Full pipeline (5-fold CV + retrain): ~3–4 hours
- Inference (150 structures): ~2 minutes on CPU
